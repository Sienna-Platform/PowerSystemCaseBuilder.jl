The `PowerSystemCaseBuilder.jl` package provides a library
of over 200 power systems test cases using the
[`PowerSystems.jl`](https://sienna-platform.github.io/PowerSystems.jl/stable/) data model.

Power system test cases are suitable for simulating and prototyping with
[`PowerSimulations.jl`](https://github.com/Sienna-Platform/PowerSimulations.jl) and
[`PowerSimulationsDynamics.jl`](https://github.com/Sienna-Platform/PowerSimulationsDynamics.jl).

For information on using the package, see the [stable documentation](https://sienna-platform.github.io/PowerSystemCaseBuilder.jl/stable/). Use the [in-development documentation](https://sienna-platform.github.io/PowerSystemCaseBuilder.jl/dev/) for the version of the documentation which contains the unreleased features.
